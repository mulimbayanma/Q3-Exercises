HTML

HyperText Markup Language

HTML, otherwise known as HyperText Markup Language, is the most common markup language for making Web pages. The layout of a Web page is described in HTML. It is also used to structure a webpage.

CSS

Cascading Style Sheets

CSS is also known as Cascading Style Sheets. It works with how HTML elements are displayed on the web. CSS can also make you control the look of a website.
